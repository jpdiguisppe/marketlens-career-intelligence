# MarketLens Demo Assets

Copy these files into the root of your local `marketlens-career-intelligence` repo.

Recommended commands from your Downloads folder after unzipping:

```bash
cp -R docs/images ~/Desktop/marketlens-career-intelligence/docs/
```

Then commit:

```bash
cd ~/Desktop/marketlens-career-intelligence
git add docs/images
git commit -m "Add MarketLens demo screenshots"
git push origin main
```

`docs/README-demo-section.md` contains a README demo section that can be pasted into README.md.
