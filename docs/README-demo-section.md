## Demo

### Job skill dashboard

MarketLens analyzes saved job postings and surfaces repeated technical skills across target roles.

![MarketLens skills dashboard](docs/images/marketlens-skills-dashboard.png)

### Resume gap analysis

Users can paste resume text, choose a target role category, and compare their current skills against the skills appearing in saved job postings.

![MarketLens resume gap analysis](docs/images/marketlens-resume-gap-analysis.png)

### Saved job postings

Imported postings are stored in the backend database and displayed in the React dashboard with extracted skills.

![MarketLens saved job postings](docs/images/marketlens-job-postings.png)

### API documentation

The FastAPI backend exposes documented endpoints for job posting import, skill extraction, grouped skill analysis, and resume analysis.

![MarketLens API docs](docs/images/marketlens-api-docs.png)
